# Игра "Лабиринт сокровищ"

Текстовая приключенческая игра, где нужно находить выход из лабиринта и решать загадки.

## Установка
```bash
make install  # или poetry install
```

## Запуск
```bash
make project  # или poetry run project
```

## Демо
[Вставьте ссылку на asciinema запись игрового процесса]

## Основные команды
- `осмотреться` - описание комнаты
- `идти [направление]` - перемещение
- `взять [предмет]` - подобрать предмет
- `инвентарь` - показать инвентарь
- `загадка` - решить загадку
- `помощь` - список команд
- `выход` - завершить игру

## Как победить?:
Добраться до комнаты treasure_room и открыть сундук помощью:

С помощью ключа (предмет treasure_key)
С помощью кода (команда solve)
